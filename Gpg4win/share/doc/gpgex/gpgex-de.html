Debugging

Falls nicht auf die GnuPG Benutzerschnittstelle zugegriffen werden kann, wird ein Nachrichtenfenster mit einer Fehlermeldung angezeigt. Um mehr Informationen zu erhalten, kann der Fehlerbericht von Kleopatra oder GPA eingesehen werden (Anleitung siehe Gpg4win-Kompendium), oder es kann eine GpgEX-Diagnosedatei angegeben werden. Dazu im Registrierungs-Editor den Schlüssel HKLM\Software\GNU\GnuPG auswählen, eine neue "Zeichenfolge" mit dem Namen GpgEX Debug File anlegen und als Wert einen Dateinamen (z.B. c:\temp\gpgex-debug-file.log) angeben. Informationen zum Protokoll werden dann nach einem Neustart von Windows an diese Datei angehangen.
